Overview – Installation – Provisioning files

Install Holo

Whenever possible, install Holo as a package. Look down on this page for a section header with your distribution's name.

From source

Alternatively, install from source by cloning the corresponding Git repos and looking at the README.md:

  • the core execuable: holo
  • optional plugins for Holo: holo-users-groups and holo-run-scripts
  • the package building tool: holo-build

Arch Linux

Packages are available from the AUR. If you prefer precompiled packages, you can use our repository (only available for x86_64 at the moment). Add the following to your /etc/pacman.conf: 

[holo]
Server = https://repo.holocm.org/archlinux/$arch

The packages are signed with GPG, so you need to import and trust the signing key:

$ sudo pacman-key -r 0xF7A9C9DC4631BD1A
$ sudo pacman-key -f 0xF7A9C9DC4631BD1A
#     check the key fingerprint before continuing, it should be:
#     2A53 49F6 B4D7 305A 85DE  D8D4 F7A9 C9DC 4631 BD1A
$ sudo pacman-key --lsign-key 0xF7A9C9DC4631BD1A

To get a list of all packages in this repo, do:

$ sudo pacman -Syl holo
